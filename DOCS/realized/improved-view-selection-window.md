# Улучшение окна выбора видов

## Что сделано

Улучшено WPF-окно выбора видов (`MainWindow`), которое открывается по кнопке «Сформировать виды».

### 1. Группировка разрезов

Четыре разреза теперь спрятаны в раскрывающийся список под главной галочкой **«Разрезы»**:
- Вид спереди
- Вид сзади
- Вид справа
- Вид слева

Галочка «Разрезы» трёхсостоятельная:
- **Включена** — все 4 разреза выбраны.
- **Выключена** — все 4 разреза сняты.
- **Серая (Indeterminate)** — выбрана часть разрезов.

При клике на главную галочку в любом состоянии происходит переключение всех разрезов.

### 2. Счётчик видов

В правой части окна добавлена информационная панель:
- **Сборок в проекте: N** — реальное количество `AssemblyInstance` в модели.
- **Будет создано видов: N × M** — произведение количества сборок на количество выбранных видов.

Счётчик обновляется динамически при изменении любой галочки.

### 3. Передача данных в окно

Команда `CreateAssemblyViewsCommand` перед открытием окна подсчитывает количество сборок и передаёт его в конструктор `MainWindow(int assemblyCount)`.

## Контрольный список ручной проверки

1. Открыть модель с несколькими сборками.
2. Нажать кнопку «Сформировать виды».
3. Убедиться, что в окне отображается:
   - галочки «План», «Разрезы», «3D вид»;
   - справа текст «Сборок в проекте: N»;
   - справа текст «Будет создано видов: N×M».
4. Убедиться, что список разрезов по умолчанию свёрнут.
5. Развернуть список разрезов и проверить наличие 4 подвидов.
6. Кликнуть по галочке «Разрезы» — все разрезы должны сняться, счётчик обновиться.
7. Кликнуть по галочке «Разрезы» ещё раз — все разрезы должны выбраться.
8. Снять одну подгалочку разреза — галочка «Разрезы» должна стать серой.
9. Кликнуть по серой галочке «Разрезы» — все разрезы должны выбраться.
10. Включить/выключить «План» и «3D вид» — счётчик должен обновляться.
11. Нажать «Сформировать» и убедиться, что создаются только выбранные виды.
